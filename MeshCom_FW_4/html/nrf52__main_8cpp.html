Config Message Parameters: Callsign, Lat, Lon, altitude Text/Pos Msg Format: LENGTH 2B - FLAG 1B - MSG xB Config Format: LENGTH 2B - FLAG 1B - LENCALL 1B - Callsign - LAT 4B(Float) - LON 4B(Float) - ALT 4B(INT)

Hello Message from Phone is: "XXaaYYzz" Config of this device is sent to phone after hello msg from phone
